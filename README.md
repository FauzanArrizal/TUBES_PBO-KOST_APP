## Latar Belakang

Perkembangan teknologi yang sangat pesat saat ini memungkinkan masyarakat untuk memanfaatkannya secara maksimal dalam berbagai aspek kehidupan. Salah satu kebutuhan yang muncul adalah kemudahan dalam mencari tempat tinggal sementara, terutama bagi mahasiswa yang menempuh studi jauh dari rumah serta pekerja kantoran yang tinggal sementara di lokasi kerja.

Kost merupakan solusi praktis dan terjangkau bagi mereka yang membutuhkan hunian sementara. Biasanya, kost banyak ditemukan di sekitar kawasan kampus dan perkantoran karena menjadi pilihan utama para mahasiswa dan pekerja yang berasal dari luar daerah.

Kondisi tersebut menginspirasi pembuatan sebuah sistem yang dapat mempermudah masyarakat dalam mencari dan mengelola tempat tinggal sementara seperti kost atau kontrakan.

Aplikasi ini dibuat dengan Java untuk mengelola data kost, termasuk kamar, penghuni, dan pembayaran.  
Project ini menggunakan MySQL sebagai database dan JDBC untuk koneksi database.

## Fitur

- Kelola kamar (CRUD)
- Kelola penghuni (CRUD)
- Kelola pembayaran (CRUD)
- Konsol interaktif untuk navigasi menu

## Teknologi

- Java 11+
- MySQL
- JDBC
- NetBeans IDE
